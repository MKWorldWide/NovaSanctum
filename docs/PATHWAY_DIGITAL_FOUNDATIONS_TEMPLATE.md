# Pathway Template: Digital Foundations

Status: Draft template (Phase 4)  
Program: NovaSanctum

## 1. Pathway Description

Digital Foundations supports learners who need baseline digital confidence for study, work, and online civic participation.

## 2. Target Learners

1. New digital learners.
2. Learners returning after long gaps.
3. Community learners preparing for technical pathways.

## 3. Entry Requirements

1. Basic reading comprehension.
2. Device access (mobile or desktop).
3. No prior technical coursework required.

## 4. Learning Outcomes

By pathway completion, learners should be able to:

1. Use web search and source evaluation basics.
2. Manage files and documents in common digital environments.
3. Apply foundational online safety and privacy practices.
4. Communicate effectively using digital collaboration tools.

## 5. Pathway Structure

1. Entry
   - Orientation to pathway expectations and study habits.
2. Foundation
   - Browser literacy, file management, account hygiene.
3. Progression
   - Productivity tools, information evaluation, practical digital workflows.
4. Applied Outcomes
   - Complete a practical digital task set (document, spreadsheet, communication exercise).
5. Transition
   - Bridge to Web and Software Fundamentals or Data Literacy.

## 6. Example Resource Categories

1. Public digital literacy guides.
2. Open tutorials on productivity software.
3. Free online safety and privacy education resources.
4. Public practice exercises for communication and collaboration tools.
